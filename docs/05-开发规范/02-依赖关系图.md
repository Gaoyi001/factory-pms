## 四、前端模块依赖

### 4.1 前端架构依赖

```
main.ts (入口)
  ├── App.vue
  ├── router/index.ts (路由)
  │     └── views/* (页面组件)
  ├── pinia (状态管理)
  └── Element Plus (UI 库)

views/* (页面组件)
  ├── api/* (API 调用)
  │     └── utils/request.ts (Axios 封装)
  ├── types/* (类型定义)
  └── element-plus (UI 组件)

utils/request.ts
  ├── axios
  ├── router (跳转登录)
  └── element-plus (消息提示)
```
