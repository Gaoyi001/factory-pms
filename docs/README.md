## 🚀 快速开始

### 后端启动

```bash
cd backend
pip install -r requirements.txt
python run.py  # 初始化数据库
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

### Docker 一键启动

```bash
cp deploy/.env.example deploy/.env
docker-compose -f deploy/docker-compose.yml up -d
docker exec -it factory-pms-backend python run.py
```

## 📋 功能模块概览

| 模块 | 描述 | 核心功能 |
|------|------|---------|
| 项目管理 | 研发项目全生命周期管理 | 项目立项、Phase-Gate 门径管理、WBS 任务分解、需求/缺陷追踪 |
| 研发实验管理 | 实验设计与数据分析 | 实验计划、参数配置、数据记录、SPC/CPK 统计分析 |
| 物料与研发 BOM | 物料与 BOM 管理 | 物料主数据、BOM 多层级编辑、ECN/ECR 变更管理 |
| 样品与试产管理 | 样品制作与试生产 | 样品登记、试产工单、检验评审、良率统计 |
| 文档与知识管理 | 技术文档与知识库 | 文档版本控制、审批工作流、在线预览、知识库 |
