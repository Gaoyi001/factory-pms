## 二、项目管理模块

### 2.1 ProjectList (项目列表)

**路径**: `/projects`  
**文件**: [project/ProjectList.vue](file:///g:/资料/code/factory-pms/frontend/src/views/project/ProjectList.vue)

**功能说明：**
- 项目列表表格展示
- 状态筛选
- 关键词搜索（名称/编号）
- 新建/编辑项目弹窗
- 删除项目（确认）
- 分页

**核心状态：**

| 状态 | 类型 | 说明 |
|------|------|------|
| `loading` | `Ref<boolean>` | 加载状态 |
| `tableData` | `{ items, total }` | 表格数据 |
| `pagination` | `{ page, page_size }` | 分页参数 |
| `filter` | `{ status, keyword }` | 筛选条件 |
| `formVisible` | `Ref<boolean>` | 弹窗显示 |
| `formMode` | `'create' \| 'edit'` | 弹窗模式 |

**核心方法：**

| 方法 | 说明 |
|------|------|
| `load()` | 加载项目列表 |
| `showForm(row?)` | 显示新建/编辑弹窗 |
| `handleSubmit()` | 提交表单 |
| `handleDelete(row)` | 删除项目 |
| `statusTag(s)` | 状态标签类型映射 |
| `statusLabel(s)` | 状态文本映射 |

**项目状态映射：**

| 值 | 文本 | 标签颜色 |
|----|------|---------|
| draft | 草稿 | info |
| active | 进行中 | success |
| on_hold | 暂停 | warning |
| completed | 已完成 | 默认 |
| cancelled | 已取消 | danger |

## 七、组件开发规范

### 7.1 Composition API

使用 Vue 3 `<script setup>` 语法糖：

```vue
<script setup lang="ts">
import { ref, reactive, computed, onMounted } from 'vue'

// 响应式数据
const count = ref(0)
const form = reactive({ name: '', email: '' })

// 计算属性
const double = computed(() => count.value * 2)

// 生命周期
onMounted(() => {
  console.log('mounted')
})
</script>
```

### 7.2 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 组件名 | PascalCase | `ProjectList.vue` |
| 组合式函数 | camelCase + use 前缀 | `useTableData()` |
| 普通变量 | camelCase | `tableData` |
| 常量 | UPPER_SNAKE_CASE | `PAGE_SIZE` |
| 事件名 | kebab-case | `@update-data` |

### 7.3 样式规范

- 使用 scoped 样式避免污染
- 推荐使用 BEM 命名法
- 公共样式放入 `style.css`
