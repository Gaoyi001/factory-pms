## 九、类型使用建议

1. **与后端对齐**：类型定义应与后端 Pydantic Schema 保持一致
2. **可选字段**：数据库允许 NULL 的字段使用 `?` 可选标记
3. **枚举类型**：有限状态值优先使用联合类型或字符串字面量类型
4. **导入方式**：

```typescript
import type { ProjectOut, TaskOut } from '@/types/project'
```
