# 目录结构

## 一、总体目录结构

```
factory-pms/
├── backend/                 # FastAPI 后端服务
│   ├── app/
│   │   ├── api/
│   │   │   ├── v1/          # v1 版本 API
│   │   │   └── __init__.py
│   │   ├── core/            # 核心模块（配置、数据库、安全）
│   │   ├── models/          # SQLAlchemy 数据模型
│   │   ├── schemas/         # Pydantic Schema
│   │   └── main.py          # FastAPI 入口
│   ├── run.py               # 初始化脚本
│   ├── requirements.txt     # Python 依赖
│   └── Dockerfile
├── frontend/                # Vue3 前端应用
│   ├── src/
│   │   ├── api/             # API 接口层
│   │   ├── views/           # 页面组件
│   │   ├── router/          # 路由配置
│   │   ├── types/           # TypeScript 类型定义
│   │   ├── utils/           # 工具函数
│   │   ├── App.vue
│   │   └── main.ts
│   ├── package.json
│   ├── vite.config.ts
│   └── Dockerfile
├── deploy/                  # 部署配置
│   ├── docker-compose.yml   # Docker Compose 编排
│   ├── .env.example         # 环境变量示例
│   └── nginx.conf           # Nginx 配置
├── docs/                    # Code Wiki 文档
└── README.md
```
